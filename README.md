# linux-procfs

Library to fetch info from `/proc` using the `*at()` system calls.

Similar libraries but lacking this functionality:
- https://github.com/giampaolo/psutil
- https://github.com/pmuller/procfs

## NOTES

  - Tested on Python 3.8+
